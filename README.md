# Avijeet Pandey Portfolio

Premium personal portfolio for Avijeet Pandey, built with React, TypeScript, Vite, Tailwind CSS, and Framer Motion.

## Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Radix UI primitives
- React Hook Form + Zod

## Development

```bash
yarn
yarn dev
```

## Build

```bash
yarn build
```

## Deploy

Deployment is handled through GitHub Pages via the workflow in `.github/workflows/deploy.yml`.
